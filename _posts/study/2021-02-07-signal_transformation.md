## 미정

#### 시간 이동(time shift)   
continuous time : $x(t) → x(t-t0)$   
discrete time : $x[n] → x[n-n0]$

#### 시간 반전(time reverse)
continuous time : $x(t) → x(-t)$   
discrete time : $x[n] → x[-n]$

#### 진폭 조정(amplitude scaling)
continuous time : $x(t) → A x(t)$
discrete time : $x[n] → A x[-n]$
